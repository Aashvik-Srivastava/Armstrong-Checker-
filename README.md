# Armstrong Number Checker 🔢

A simple Python CLI project to check Armstrong numbers and display Armstrong numbers in a given range.

## 🚀 Features
- Check single number
- Find Armstrong numbers in range
- Menu-based system

## 🛠️ Tech Stack
- Python

## 🧠 Example
153 → Armstrong number ✔️  
370 → Armstrong number ✔️  

## ▶️ Run
python armstrong.py
